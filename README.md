# Automated-fog-water-collector
ESP32-based automated fog water collection system developed for isotopic characterization of fog water at IIT Kanpur.

This repository contains the ESP32-based automation code, circuit design, and PCB layout developed for the automated fog water collection system used in the MTech thesis:

“Isotopic Characterization of Fog Water”

Department of Civil Engineering (Hydraulics and Water Resources Engineering)
Indian Institute of Technology Kanpur

Features:
Real-time visibility monitoring
Automated curtain control
MQTT sensor communication
Watchdog reset logic
Motor driver control
Data logging

Hardware Used:
ESP32 microcontroller
Biral visibility sensor
Rhino DC geared motor
Proximity sensors
PCB-based control system
